# Chat Room

A simple messenger WebApp made with Flask.
It includes :
- Registration
- Login
- Chat UI

Storage solution: Sqlite3

## Installation

1. Install Docker and docker-compose
2. Clone repo

## Usage

Open terminal under the repo. 
`$ docker-compose up`
